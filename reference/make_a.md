# Convert an `Spde` object to a A object

Convert an `Spde` object to a A object

## Usage

``` r
make_a(object, data)

# S4 method for class 'Spde'
make_a(object, data)
```

## Arguments

- object:

  The `Spde` object

- data:

  The `data.frame` to combine with the `Spde` object to create the `A`
  matrix.
