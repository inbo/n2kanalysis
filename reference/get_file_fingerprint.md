# Get the file fingerprint of an `n2kModel` object

Get the file fingerprint of an `n2kModel` object

## Usage

``` r
get_file_fingerprint(x)

# S4 method for class 'n2kAnalysisMetadata'
get_file_fingerprint(x)

# S4 method for class 'n2kManifest'
get_file_fingerprint(x)
```

## Arguments

- x:

  the `n2kModel` object

## Value

the file fingerprint of the object
